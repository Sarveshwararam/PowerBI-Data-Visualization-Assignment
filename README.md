# Power BI Data Visualization Assignment

## Overview
This project demonstrates the use of **Microsoft Power BI** for building interactive and insightful dashboards.  
The report focuses on analyzing key metrics, applying data transformations, and presenting results through visually engaging charts and reports.

## Features
- Interactive Power BI dashboard with filters and slicers  
- Data cleaning and transformation using Power Query  
- DAX measures and calculated columns for KPIs  
- Dynamic visuals for trend and performance analysis  
- Professionally formatted and color-coded report pages  

## Tools Used
- **Power BI Desktop**
- **Power Query**
- **DAX (Data Analysis Expressions)**

## How to Use
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop**.
3. Explore the report pages and interact with visuals using filters and slicers.
